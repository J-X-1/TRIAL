## Running the Evaluation

For each question, run:
```
python trial_qa.py "QUESTION TEXT HERE"
```

Then check:
1. **Did it retrieve the right sections?** Compare "Retrieved Sections" to the expected sections above.
2. **Is the answer substantively correct?** Compare the answer to the expected answer above.
3. **Did citation validation pass?** Check for hallucinated citations.
4. **For out-of-scope questions:** Did it correctly decline to answer?

It is not the case that you give 1 point for each of these.

Score each question as PASS (4/4), PARTIAL(1-3/4), or FAIL(0/4).
